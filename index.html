Hi! It's a free app for creating screenshots.

How to use:

  • Click the right mouse button for calling a context menu.
  • Clicking on "New" is for creating a new screenshot.
  • Choose "Crop" for cropping an area.
  • "Ctrl+Z" allows you to cancel(undo), and "Ctrl+Shift+Z" to return(redo).
  • For focusing on a particular area use "Rect".
  • Take "Arrow" for focusing on details.
  • Write something with "Text" tool.
  • Press `ESC` or right mouse button to choose default instrument.
  • Press "Ctrl+S" to save your screenshot.
    By default it will be send to server and copied to buffer.
  • To see all shortcuts press "F2" or open "More->Shortcuts".

Shortcuts

  • New - "Ctrl + N"
  • Save - "Ctrl + S"
  • Restart - "Ctrl + R"
  • Quit - "Ctrl + Q"
  • Undo - "Ctrl + Z"
  • Redo - "Ctrl + Shift + Z"
  • Zoom in - "Ctrl + Shift + Plus"
  • Zoom out - "Ctrl + Minus"
  • Defaul zoom - "Ctrl + 0"
  • Default tool - "D"
  • Crop - "C"
  • Arrow - "A"
  • Rect - "R"
  • Pen - "P"
  • Help - "F1"
  • Shortcuts - "F2"
  • Settings - "S"
  • Emoji - "Alt + `Emoji letter`"
  • Blur - "F"
  • Text - "T"
